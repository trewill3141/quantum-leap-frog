### PPC Ping Tree Keyword Models

## Company 
BorrowWorks 

## Client
All 5 Portfolios

## Problem 
Client required custom end-to-end operational and analytical data feeds to support ad buying for affiliates, but lacked automated keyword recommendation and bidding systems that could integrate with affiliate software platforms and advertising APIs, resulting in manual keyword management and suboptimal ad spend allocation across affiliate networks.

## Solution
Built custom end-to-end operational and analytical data feeds supporting ad buying for affiliates with integrated keyword recommendation engine using SageMaker, Google Ads API, and affiliate software API. Developed automated workflows using Airflow that orchestrate data collection, keyword analysis, bid optimization, and performance monitoring.

Architected keyword recommendation engine using AWS SageMaker (machine learning models) that analyzes historical performance data, keyword attributes, and market trends to recommend optimal keywords for affiliate campaigns. Integrated Google Ads API for automated keyword research, bid management, and campaign optimization based on ML-driven recommendations.

Built data pipelines using Airflow and webhooks to collect real-time performance data from affiliate software platforms, transforming and loading into Snowflake for analysis. Developed operational feeds that provide near real-time keyword performance metrics and analytical feeds that support strategic decision-making through comprehensive reporting.

All keyword recommendations are automated and integrated with affiliate software APIs, enabling seamless deployment of ML-driven keyword strategies across affiliate networks with automated bid adjustments based on performance data.
